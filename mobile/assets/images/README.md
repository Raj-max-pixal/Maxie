## Image Specifications

### App Icons
- **Adaptive Icon**: 512x512px (foreground), 512x512px (background)
- **Launcher Icon**: 192x192px (mdpi), 144x144px (hdpi), 96x96px (xhdpi), 72x72px (xxhdpi), 48x48px (xxxhdpi)
- **Play Store**: 512x512px

### UI Images
- **Format**: PNG with transparency
- **Resolution**: 2x or 3x for retina displays
- **Color Profile**: sRGB
