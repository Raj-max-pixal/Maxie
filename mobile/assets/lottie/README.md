# Lottie Animations Directory

This directory will contain all Lottie animation files for MAXie.

## Required Files

Place your Lottie JSON files in this directory. See `ANIMATION_GUIDE.md` for the complete list of required animations.

## Quick Start

1. Create or download Lottie animations
2. Place `.json` files in this directory
3. Update `pubspec.yaml` to include assets
4. Use in Flutter code with `Lottie.asset()`

## Example Usage

```dart
Lottie.asset(
  'assets/lottie/maxie_happy.json',
  width: 200,
  height: 200,
  fit: BoxFit.contain,
)
```
