Yes. MAXie is currently available as a free early project build.

MAXie is designed local-first, with offline-friendly companion features.

Which operating systems are supported?

Windows desktop is supported, and Android is available as a test APK.

Is my data private?

MAXie is built around local storage and user-controlled data flows.

How do I report bugs?

Open an issue on GitHub with your device, OS version, and steps to reproduce.

Use GitHub issues or discussions to share feature ideas and roadmap suggestions.
