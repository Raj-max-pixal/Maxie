# Privacy Policy

MAXie is designed to be local-first.

## Data Stored Locally

MAXie stores settings, pet position, appearance preferences, memories,
friendship/XP, inventory, needs, achievements, and stories on the user's device.

## Active App Awareness

If enabled, MAXie reads the active application name and window title so it can
react to apps like VS Code, Spotify, YouTube, Discord, and browsers. MAXie does
not capture keystrokes or read file contents.

## AI Providers

Ollama runs locally. Gemini and OpenAI-compatible providers send chat messages
to the configured provider only when the user uses AI chat or AI features that
require a provider.

## Images

Uploaded pet images are processed locally in the settings window. They are not
uploaded by MAXie.

## Notifications and Startup

Notifications and Windows startup are opt-in settings.
