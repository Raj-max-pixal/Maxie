# Roadmap

## Before Public Launch

- Replace placeholder GIF files with a consistent production sprite pack.
- Add final application icon, tray icon, installer icon, and website screenshots.
- Test installer on a clean Windows user account.
- Run privacy review for active-app awareness and AI provider settings.
- Add signed installer and code-signing workflow if distributing broadly.

## Version 1.0

- Stable pet window, settings, movement, reactions, state persistence, local AI
  configuration, and installer.

## Version 1.5

- Polished sprite animations, sound effects, better home/world moments, and
  stricter performance tuning.

## Version 2.0

- Local plugin packs, advanced character creator, richer multi-pet behavior, and
  optional cloud sync only after privacy review.
